
# How do I use the concatenation operator **&**?

The **&** operator is used to concatenate multiple strings into one column. You can use it to concatenate any combination of source file fields and constants. Lookup fields, previous column references (COL.nn), and the PRIOR() function are not supported.

The **&** operator can only be used in **Extract Column Logic** text.
